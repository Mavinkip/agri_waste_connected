## 📖 Overview

Agri-Waste Connect is a multi-role Flutter application designed to streamline agricultural waste management.

---

## ✨ Features

### 👨‍🌾 Farmer App
- Multi-step waste selling wizard
- Earnings tracking
- Simple guided UX

### 🚛 Driver App
- Route-based pickups
- Offline-first support
- Live payout calculation

### 🏭 Admin Panel
- Fleet management
- Price control system
- Inventory tracking
